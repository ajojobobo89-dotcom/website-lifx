[![npm version](https://badge.fury.io/js/web-downloader.svg)](https://badge.fury.io/js/web-downloader)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Library Node.js untuk mendownload konten web dengan mudah. Mendukung download HTML, file, dan multiple URLs.

## Fitur

- ✨ Download HTML dengan parsing otomatis
- 📁 Download file ke disk
- 🔗 Ekstrak links dan images dari HTML
- ⚡ Download multiple URLs secara parallel
- 🛡️ Custom headers dan timeout
- 📊 Error handling yang baik

## Instalasi

```bash
npm install web-downloader
