📥 Web Downloader

Library Node.js untuk mendownload konten web dengan mudah

GitHub Instalasi

Instalasi

npm install website-lifx

Contoh Penggunaan

const WebDownloader = require('website-lifx');
const downloader = new WebDownloader();

// Download HTML
downloader.downloadHTML('https://example.com')
    .then(result => console.log(result.title))
    .catch(error => console.error(error));

Fitur

  • ✅ Download HTML dengan parsing otomatis
  • ✅ Download file ke disk
  • ✅ Ekstrak links dan images
  • ✅ Multiple URLs parallel
  • ✅ Custom headers & timeout
